## Kāñchī 23 Jagadguru Śrī Sachchitsukhendra Sarasvatī Ārādhanā
काञ्ची २३ जगद्गुरु श्री सच्चित्सुखेन्द्र सरस्वती आराधना is observed on Śukla-Saptamī tithi of Vṛṣabha (solar) month (madhyaahna/puurvaviddha).

## dattAtrEya jayantI
दत्तात्रेय जयन्ती is observed on Paurṇamāsī tithi of Mārgaśīrṣa (lunar) month (pradosha/puurvaviddha).



```
आदौ ब्रह्मा मध्ये विष्णुरन्ते देवः सदाशिवः।
मूर्तित्रयस्वरूपाय दत्तात्रेय नमोऽस्तु ते॥
```

## Kāñchī 29 Jagadguru Śrī Pūrṇabodhendra Sarasvatī Ārādhanā
काञ्ची २९ जगद्गुरु श्री पूर्णबोधेन्द्र सरस्वती आराधना is observed on Śukla-Ekādaśī tithi of Śravaṇa (lunar) month (madhyaahna/puurvaviddha).

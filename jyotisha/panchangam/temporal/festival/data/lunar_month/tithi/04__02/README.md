## jagannAtha-ratha-yAtrA
जगन्नाथ-रथ-यात्रा is observed on Śukla-Dvitīyā tithi of Āṣāḍha (lunar) month (sunrise (default)/puurvaviddha (default)).

## amRtalakSmI-vratam
